# Teste Globo

Teste de avaliação para processo seletivo da Globo.

## Instruções

1. Configurando ambiente local para desenvolvimento;
2. Executar testes;
3. Executar aplicação.

### Configurando ambiente local para desenvolvimento

```shell
make install
```

### Executar testes

```shell
make test
```

Coverage

```shell
make coverage
```

Sonar Qube

```shell
make sonar-scan
```

### Executar aplicação

```shell
make run
```
