# Teste GLobo

Teste de avaliação para processo seletivo da Globo.

##  Instruções

1. Configurando ambiente local para desenvolvimento;
2. Executar testes;
3. Executar aplicação.

### Configurando ambiente local para desenvolvimento

```shell
make install
```

### Executar testes

```shell
make test
```

Coverage

```shell
make coverage
```

### Executar aplicação

```shell
make run
```
